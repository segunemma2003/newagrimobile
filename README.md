# Learn with Agrisiti

Learn with Agrisiti - Mobile Learning Application for Agrisiti Academy.

## Overview

Learn with Agrisiti is a comprehensive mobile learning platform that provides agricultural education through courses, modules, assignments, and interactive learning experiences.

## Features

- User authentication and profile management
- Course browsing and enrollment
- Progress tracking
- Assignments and submissions
- Tests and quizzes
- Notes and comments
- Certificates
- Messaging (Coming Soon)

## Requirements

* Flutter >= 3.24.0
* Dart >= 3.4.0

## Getting Started

```bash
# Clone the repository
git clone <repository-url>

# Install dependencies
flutter pub get

# Run the app
flutter run
```

## Configuration

1. Copy `.env.example` to `.env`
2. Update API base URL and other configuration values
3. Configure Android and iOS app identifiers

## Build

### Android
```bash
flutter build apk --release
```

### iOS
```bash
flutter build ios --release
```
